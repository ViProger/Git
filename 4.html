Основы работы с языком CSS

Изучива стандартные и основные теги HTML мы сделали основную структуру сайта, а вот что же с красотой, ведь сайт должен привлекать внимание и быть приятным внешне. За это отвечает язык стилей CSS

Как бравило команды (стили) Хранятся в отдельном файле и подлкючаются к страницам с помощтю следующей конструкции:

<link rel="stylesheet" href="имяФайла.css">

Как работать с CSS

Каждому тегу из HTML соответствует одноименный селектор в CSS. Например чтобы обратится ко всем обзацам и покрасить их в один цвет нужно обратится к селектору р. Следом за селектором ставятся фигурные скобки внутри которых находятся свойства отвечают за, собствнно свойства обекта, тоесть цвет, размер, форма, и прочее. В своем время свойства имеют свои значения, например свойство цвет имеет значение красный. Выглядит это следующим образом

p { color: red; }

Помимо ключевого слова red можно использовать и другие цвета на английсокм языке.

Способы задания цветов в CSS

Помимо английских слов есть и другие способы задания цвета, так как ключивыми словами не получится сделать всевозможные оттенки и цвета.

Первый способ задания цвета RGB

p { color: rgb(255, 236, 114); }

Второй способ задания цвета #

p { color: #FF2547; }

Жирность текста на CSS

В данном уроке мы с вами изучим свойство font-weight, которое позволяет сделать текст жирным или наоборот - отменить жирность (к примеру, для заголовков, которые жирные по умолчанию). Чтобы сделать текст жирным, следует указать значение bold, а чтобы отменить жирность - значение normal. Это свойство работает аналогично тегу b, который мы с вами уже разобрали. Разница в том, что через CSS управлять жирностью гораздо удобнее - я могу заставить все абзацы стать жирными, а все заголовки - нежирными, сделав всего пару записей в CSS файле. Межстрочный интервал на CSS

Давайте теперь научимся задавать межстрочный интервал тексту. Под межстрочным интервалом подразумевается расстояние между линиями текста, то есть белый промежуток между ними. Для этого предназначено свойство line-height, задающее высоту линии текста. При использовании свойства line-height вас может ожидать некоторый подвох: это свойство не задает промежуток между строками текста, как могло бы показаться, а задает высоту линии текста. Это значит, что реальный видимый промежуток между строками будет вычисляться так: line-height - font-size = видимое расстояние между строками текста. В следующем примере расстояние между строками текста будет 50px - 20px = 30px:

<p> длинный текст... </p> p { font-size: 20px; line-height: 50px; text-align: justify; width: 400px; }

Значение line-height без единиц

Значением свойства line-height не обязательно должно служить число в некоторых единицах. Можно также просто написать число или дробь. В этом случае настоящее значение line-height можно будет найти, умножив его на значение font-size. К примеру, font-size равен 10px, а line-height - 1.5. В этом случае реальное значение line-height будет 10px * 1.5 = 15px. Ну, а видимый белый промежуток между линиями текста будет 5px: 15px - 10px = 5px. Преимущество такого способа задания line-height в том, что при изменении размера шрифта автоматически будет меняться и межстрочный интервал. Посмотрим применение описанного на примере:

<p> длинный текст... </p> p { font-size: 20px; line-height: 1.5; text-align: justify; width: 400px; }

Свойство-сокращение FONT

Ранее мы рассмотрели множество свойств отвечающих за отображение шрифта, семейство, жирность, курсив и прочее. ДЛя упрощения работы с данными свойствами в CSS существует свойство-сокращение FONT. Оно нужно например чтобы задать всем параграфам на сайте или странице одинаковый стиль.

Синтаксис данного свойства следующий:

КУРСИВ ЖИРНОСТЬ РАЗМЕР/Межстрочный_ИНТЕРВАЛ семейство

При этом важно учесть порядок написания значений, А так же обязательными значениями являются "размер шрифта" и "Семейство". Так же следует учесть, что другие значения кроме описаных выше не входят в данное сокращение.

Например следующие записи равнозначны друг другу. Но ведь вторая проще и компактнее.

p { font-size: 16px; font-family: Arial; } Или p { Font: 16px Arial; } p { font-size: 16px; font-family: Arial; line-height: 50px; } Или p { Font: 16px/50px Arial; } p { font-size: 16px; font-family: Arial; font-weight: bold; } Или p { Font: bold 16px Arial; }

Практические задания

Сократите код, используя свойство-сокращение font:

p { font-family: "Times New Roman"; font-size: 13px; line-height: 20px; }

Результат

p { font: 13px/20px "Times New Roman"; }

Сократите код, используя свойство-сокращение font:

p { width: 300px; font-family: Arial; color: red; font-size: 40px; font-weight: bold; }

Результат

p { width: 300px; color: red; font:Bold 40px Arial; }

Сократите код, используя свойство-сокращение font:

p { font-family: Arial; font-size: 40px; font-weight: bold; text-indent: 50px; font-style: italic; line-height: 60px; }

Результат

p { text-indent: 50px; font:Bold italic 40px/60px Arial; }

Эффекты текста на CSS

Свойство text-decoration позволяет задавать некоторые эффекты для текста: подчеркивание, перечеркивание, линию сверху, а также отменять такие эффекты, если какой-либо тег имеет их по умолчанию. Чаще всего это свойство используются для отмены подчеркивания ссылок (они по умолчанию подчеркнуты)

Свойство text-decoration может принемать следующие значения:


Overline - линия сверху

Underline - линия снизу

line-through - линия зачеркивает текст

none - Значение убирает все эффекты
