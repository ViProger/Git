Теги ol и ul

Эти теги как мы помним создают списки. Ol - НЕ упорядоченные списки. ul - упорядоченные списки.

Ссылки

Ссылка создается с помощью тега a. В этом теге обязательно должен быть атрибут href, в который следует записывать адрес той страницы, на которую ведет ссылка

Изображения на HTML

Для этого предназначен тег img, имеющий обязательный атрибут src, в котором следует задавать путь к файлу картинки. При этом сам тег не требует закрывающего тега.

Таблицы

Таблицы в html создаются по рядам. Код имеет жесткую структуру главный тег table внутри него лежат тег tr которые создают ряды в уже внутри рядов находятся теги td которые создают ячейки. Так же есть теги которые создают Заголовки для таблицы, они так же размещаются внутри строки (тега tr) и называются th

Значение Описание
table Образующий таблицу тег
tr Создает строки
td Создает ячейку внутри строки
th Создает заголовки внутри строки

RGB Вариант похожий на использование #, его так же можно получить используя спец программу или онлайн сервис. конвертера. Синтактсис след: color:rgb(255, 236, 114);

Жирность текста на CSS

Задать жирность текста можно не только обернув его в тег b, но еще и присвоив свойству font-weight одно изщ следующих значенией

Курсив текста на CSS

Задать курсив текста можно не только обернув его в тег i, но еще и присвоив свойству font-style значение italic, для отмены использовать значение normal.

Размер текста на CSS

Задать размер текста можно не только обернув его в один из тегов h1, h2, h3, h4, h5, h6, но еще и присвоив свойству font-size значение в пикселях или например pt что более привычно для нас из программы Ворд, pt это размер кегли применяемой к тексту.

Если название шрифта состоит из более чем 1 слова то его необходимо взять в ковычки

Межстрочный интервал на CSS

line-height задает растояние между строками текста (межстрочный интервал) в пикселях. Следует учесть тот факт, что это свойство не задает промежуток между строками текста, как могло бы показаться, а задает высоту линии текста. Это значит, что реальный видимый промежуток между строками будет вычисляться так: line-height - font-size = видимое расстояние между строками текста.

Так же если указать значение свойсва line-heihgt без единиц измерения, то произойдет умножение font-size на указанное значение.

К примеру, font-size равен 10px, а line-height - 1.5. В этом случае реальное значение line-height будет 10px * 1.5 = 15px. Ну, а видимый белый промежуток между линиями текста будет 5px: 15px - 10px = 5px.

курсив жирность размер_шрифта / межстрочный_интервал семейство

Красная строка в тексте

Красная строка задается с помощью свойства TEXT-INDENT, в качестве единиц измерения используется чаще всего px или же %

Text-decoration

Text-decoration задает НЕКОТОРЫЕ эффекты для текста, например отменяет подчеркивание ссылок или убирает маркеры у маркеровоного меню.

Значение Описание
underline подчеркивание текста
overline надчеркивание текста
line-through зачеркивание текста
