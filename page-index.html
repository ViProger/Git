Выравнивание блочных элементов

Свойство margin используется не только для задания отступов, но так же и для центрирования БЛОЧНЫХ элементов, для этого правый и левый отступ следует задать в значение auto. margin: 0px auto. Следует отметить, что этот способ действует только на БЛОЧНЫЕ элементы, которым задана ширена, а так же только по горизонтали. Значение auto для margin на самом деле может не только центрировать. Фактически это значение позволяет рассчитать отступ автоматически и сделать его максимально возможным. Когда мы ставим auto для левого и правого отступа - элемент центрируется. Можно, однако, поставить только левый отступ. В этом случае элемент получит максимально возможный отступ слева и займет крайнее правое положение. При этом не запрещено указать отступ справа для данного элемента, что просто не даст ем прилипнуть к правому краю слишком сильно.

Выравнивание строчных элементов

Строчные элементы выравниваются не так как БЛОЧНЫЕ. ДЛя выравнивания строчных элементов используется свойство text-align, которое присваивается родительскому элементу. То есть, если родителю задано text-align: centerl; то все потомки будут позиционироваться по центру. Выравнивание строчных элемнтов делается с помощью использования общепринятых слов left, right, center.

Выравнивание строчно-блочных элементов

Строчно-блочные элементы выравниваются так же как и строчные - с помощью text-align в нужном значении, заданном для родителя

Флекс элементы на CSS

Кроме блочного типа элементов есть еще один тип - флекс элементы. Для получения таких элементов необходимо РОДИТЕЛЮ задать свойство dispalay со значением flex. При этом сам элемент остается блочным а вот все го ПОТОМКИ становятся флекс элементами.

Флекс элементы подобны блочным, имеют ширену и высоту, но при этом они выстраиваются в ряд внутри своего РОДИТЕЛЯ. Если высота родителя не указана она растягивается по ширене потомков (словно Строчно-блочный элемент)

Если РОДИТЕЛЮ не задана ширена, то он займет всю доступную ширену страницы

Строчно-блочный родитель

Если родителю задать значение display: inline-flex; то родитель будет вести себя как строчно-блочный элемент. В таком случае ширена родителя будет зависит от ширены потомком.

Выравнивание Flex элементов

Для выравнивания флексов используется свойство justify-content. Оно принимает много значений, таблицу с которыми вы увидите далее

Значение Описание
Center Выравнивает элементы по центру
Space-between Растягивает элементы по всей ширене с равномерными отступами. При этом первый элемент прижат к левой стороне а последний к правой
Space-around Рястягивает элементы по всей ширене с равными отступами со всех сторон. При этом первый и последний элементы не прилипают к краям. НЕ путать с space-between, в котором крайние элементы прилипают к краям
space-evenly Тоже похожее выравнивание, но в данном случае уже, все отступы между элементами равны со всемх сторон, и с краев и между элементами

Абсолютное позиционирование элементов на CSS

Абсолютное позиционирование - элемент размещается на странице невзирая на присутствие других элементов по указанным координатам отсчитаным от левого верхнего угла браузера (экрана). При этом другие элементы в свою очередь веут себя так будто абсолютно позиционируемого объекта не существует (то есть они не двигаются никак он просто накладывается на них)

position: absolute;

Так же для позиционирования необходимо указать отступы сверху, снизу, справа, слева (top, bottom, right,left)

Относительное позиционирование элементов

Относительное позиционирование - сдвиг элемента относительно текущего положения, при этом все другие элементы воспринимают его так, буд-то он и не сдвигался (как будто он стоит слева сверху он объект находится по центру перекрывая другой текст например, так как тест думаюет что обект стоит слева сверху)

position: relative;

Смещение указывается с помощью отступов сверху, снизу, справа, слева (top, bottom, right,left).

А так же если родителю не указать относительное позиционирование, то потомок будет позиционироваться относительно окна браузера!!!

Давайте сделаем светофор


В предыдущем уроке мы с вами разобрали, что, если родителю задать relative, то его потомки будут абсолютно позиционироваться относительно него. Так чаще всего поступают, когда не хотят, чтобы родительский элемент выпал из нормального потока. Может быть и такое, что у родителя задан absolute. В этом случае потомки с absolute также будут позиционироваться относительно такого родителя, а не относительно окна браузера:

Если элементу задать просто абсолютное позиционирование и не указать координаты, то он просто останется на месте, при этом все остальные обекты просто наедут на него сверху и все!!!

Центрирование через абсолютное позиционирование

С помощью абсолютного позиционирования можно центрировать объект, для этого нужно задать равный отступ со всех четрых сторон.

Отрицательные значения при позиционировании

Изученные нами свойства top, right, bottom и left могут принимать не только положительные, но и отрицательные значения. Это значит что если родительский объект стоит по центру, и потомки имеют отрицательное позиционирование, то они просто вылезут наружу родительского объекта

Фиксированное позиционирование

Фиксированное позиционирование работает примерно так же как и абсюлютное, Но если в случае с абсолютным позиционированием объект остаеться на месте и при прокрутке страницы листается вместе с содержимым, то фиксированное двигается вместе с прокруткой. (например меню которое не пропадает пока вы читаете контент)

Наложение объектов по оси Z

По умолчанию, если элемент идет ниже в HTML, то и по оси Z он будет ниже тех что идут выше, тоесть он будет подкладываться под другие.

Регулировка положения

Чтобы отрегулированть порядок наложения используется свойство z-index, прнимающее целые положительные или отрицательные значения. Влияет только на элементы которые спозиционированны как absolute, fixed, relative. Принцип работы данного свойства следующий - чем больше значение тем выше объект.

Если объекту не задан индекс, то он по умолчанию равен нулю (0)

Значение Описание
left Отмена обтекания слева
right Отмена обтекания справа
both отмена обтекания по двум сторонам
